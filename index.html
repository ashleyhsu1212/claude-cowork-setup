Prerequisites
Before You Begin

Make sure you have all of the following before starting:

  • Paid Claude subscription — Pro, Max, Team, or Enterprise plan
  • Compatible computer — Mac (macOS 11+) or Windows 10+
  • Google Chrome or Microsoft Edge — required for the browser extension
  • Stable internet connection — first launch downloads ~2GB of files
⚠️ Brave, Arc, and other Chromium-based browsers are not currently supported for the Claude in Chrome extension.
1
Step 1
Download & Install the Desktop App
  • Go to claude.com/download and download the installer for your OS
  • Mac: Open the .dmg file and drag Claude to your Applications folder
  • Windows: Run the .exe installer and follow the prompts using the default path
  • Launch Claude — it will download ~2GB of files on first boot. Wait for it to complete
💡 The installer itself is small (~7MB). The bulk of the download happens after launch, so keep the app open until setup finishes.
2
Step 2
Sign In & Open Cowork Mode
  • Open the Claude Desktop app and sign in with your Claude or Google account
  • Configure your keyboard shortcut and menu bar launcher when prompted, then click Continue
  • Click the Cowork tab at the top of the app to switch from chat into Cowork mode
✅ You're now in Cowork mode. You can describe any multi-step task and Claude will begin planning and executing it autonomously.
3
Step 3
Connect the Chrome Browser Extension

Pairing Cowork with Claude in Chrome lets Claude automate browser tasks — web navigation, form filling, data extraction — alongside your local files.

1
Install the extension Go to claude.com/claude-for-chrome and click "Add to Chrome"
2
Sign in to the extension Use the same Claude account credentials when prompted
3
Pin the extension Click the puzzle piece 🧩 icon in Chrome → click the thumbtack next to "Claude"
4
Enable in Desktop app Click your initials (lower left) → Settings → find Claude in Chrome → toggle it on
5
Activate per conversation In any Cowork chat, use the Connectors dropdown to enable Claude in Chrome for that session
⚠️ If the extension isn't detected after installing, restart Chrome so it picks up the native messaging configuration.
4
Step 4
Add Connectors

Connectors give Claude live access to your apps — Slack, Notion, Google Drive, Linear, and hundreds more.

Browse & Install
1
Open Connectors Click your initials → Settings → Connectors → Browse Connectors
2
Choose your type Web = browser-based APIs  |  Desktop = locally installed with deeper system access
3
Authorize Click a service (e.g., Slack, Notion) and complete the OAuth authorization flow
Add a Custom Connector
1
Go to Customize → Connectors Click the "+" button
2
Enter connector details Provide the connector name, URL, and optional OAuth credentials (Client ID + Secret)
3
Click "Add" and connect Follow the same authorization flow as catalog connectors

Install from the Library
1
Switch to Cowork mode Click the Cowork tab in the Claude Desktop app
2
Open Customize Click "Customize" in the left sidebar — this brings together all your plugins, skills, and connectors
3
Browse Plugins Click "Browse Plugins" to open the catalog modal
4
Install your plugin Find the plugin that matches your role (e.g., Marketing) and click "Install"
5
Use your new skills Type "/" or click "+" in any Cowork chat to see the skills your plugin unlocked
Customize an Installed Plugin
1
Go to Customize → Plugins Find your installed plugin in the list
2
Click "Customize" Located in the upper right corner of the plugin card
3
Chat with Claude A new Cowork task opens where Claude walks you through tailoring the plugin to your workflow, tone, and connected tools
Install a Custom / Shared Plugin
1
Go to Customize → Plugins Click "Upload Plugin"
2
Select your .plugin file This could be shared by a colleague or downloaded from the Anthropic GitHub repo
3
Follow setup prompts Connect any required services and you're ready to go
Build Your Own Plugin
1
Install "Plugin Creator" Find and install this built-in plugin from the Browse Plugins catalog
2
Follow the guided flow Claude walks you through naming, selecting skills, and connecting the right services
3
Share with your team Export as a .plugin file and share it — teammates install it in seconds with Upload Plugin
✅ Team/Enterprise admins: You can push plugins to all members via Admin Settings → Manage Plugins — no manual install needed per person.
6
Step 6
Tips for Best Results
  • Be specific with task descriptions — the more context you provide, the better and faster Claude completes your task
  • Set standing instructions — go to Settings to specify your role, preferred tone, and output format so Claude applies them to every session automatically
  • Keep the app open — closing the Desktop app ends any active Cowork session in progress
  • Watch your usage — Cowork tasks consume more of your plan allocation than regular chat since they're multi-step and compute-intensive
